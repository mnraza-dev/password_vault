# 🔐 Password Vault

A simple, secure password manager built with React (Next.js), TailwindCSS, and local storage. Encrypt your passwords with a master key, and manage them with ease.

## 🚀 Features

- ✅ Master password to unlock the vault
- 🔐 AES encryption of stored passwords
- 🗃️ Add, search, and delete entries
- 👀 Toggle visibility of passwords
- 📋 Copy to clipboard
- 🧠 Persistence using `localStorage`
- 💡 Clean and responsive UI with Tailwind

## 📦 Tech Stack

- React (with Next.js)
- TypeScript
- TailwindCSS
- CryptoJS (for AES encryption)
- Heroicons
